Usage

Step 1: Import the style

  • Using the CLI:
unistylus add "components/button"
  • Or, include SCSS:
@use '@unistylus/bootstrap/components/button';
  • Or, include CSS:
<link rel="stylesheet" href="https://unpkg.com/@unistylus/bootstrap-css@0.0.11/components/button.css">

Step 2: Use the class

<button class="button">button</button>

Usage

Step 1: Import the style

  • Using the CLI:
unistylus add "components/icon-success-6x"
  • Or, include SCSS:
@use '@unistylus/bootstrap/components/icon-success-6x';
  • Or, include CSS:
<link rel="stylesheet" href="https://unpkg.com/@unistylus/bootstrap-css@0.0.11/components/icon-success-6x.css">

Step 2: Use the class

<i class="icon-success-6x icon-github"></i>

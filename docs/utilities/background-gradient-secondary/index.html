background-gradient-secondary
Usage

Step 1: Import the style

  • Using the CLI:
unistylus add "utilities/background-gradient-secondary"
  • Or, include SCSS:
@use '@unistylus/bootstrap/utilities/background-gradient-secondary';
  • Or, include CSS:
<link rel="stylesheet" href="https://unpkg.com/@unistylus/bootstrap-css@0.0.11/utilities/background-gradient-secondary.css">

Step 2: Use the class

<div class="background-gradient-secondary" style="padding:1rem;width:100%;max-width:300px;">background-gradient-secondary</div>
